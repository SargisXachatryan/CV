# Sargis Khachatryan — CV & Portfolio

Personal CV and project portfolio, built with React + TypeScript + Vite and deployed on GitHub Pages.

**[→ View CV](https://sargisxachatryan.github.io/CV/)**　　**[→ View Portfolio](https://sargisxachatryan.github.io/CV/portfolio)**

## Stack

React · TypeScript · Vite · CSS · GitHub Pages

## Local dev

```bash
npm install
npm run dev
```

## Deploy

Pushing to `main` triggers the GitHub Actions workflow which builds and deploys to GitHub Pages automatically.
